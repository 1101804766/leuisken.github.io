今天在配置git的时候有几条命令忘了，在这里记一下

git config --global user.name <"YouName">
	git config --global user.email <"email">
	ssh-keygen -t rsa -C <"email">

顺便说一下，git bash的粘贴在命令窗口的标题，右键在编辑里找。或者选属性，里面有个快速编辑模式，打勾的话默认右键就是粘贴了，但怎么说呢，我用着觉得有点变扭的说。

git remote add origin <ssh地址>

git remote set-url origin <ssh地址>

至于说ren这个命令，本身没什么特别的，但和通配符*结合在一起就强大了。比如

ren 123*.jpg 456*.jpg

懒得举例子了，大家自己动手试试，实践出真知。
